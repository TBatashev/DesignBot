1) Заполняем в файле configs.py все нужные данные
2) Запускаем файл main.py


Команды для админов:
/admin - админ панель
/add - добавить ТЗ в общий списо

Команды для юзеров:
/start 
Все остальное в колбэк-дате


Логика бота:
1)Человек получает ТЗ для выполнения задания, перед этим проверяется подписка на ваш канал
2) Юзер отправляет готовую работу на проверку. Всего попыток 2 на 10 минут
3) Работа отправляется в админский чат, где админы принимают решения принять или нет
